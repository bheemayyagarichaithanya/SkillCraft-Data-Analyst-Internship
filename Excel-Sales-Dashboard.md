# Task 01 – Excel Sales Dashboard 📊

## Objective

Analyze Superstore Sales Data and create an interactive dashboard using Microsoft Excel.

## Activities Performed

* Cleaned and organized sales data
* Created Pivot Tables for analysis
* Built charts for sales trends and category performance
* Applied Conditional Formatting to identify high and low performers

## Key Insights

* Identified top-performing product categories
* Compared sales and profit across categories
* Analyzed monthly sales trends

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Conditional Formatting

## Deliverables

* Excel Dashboard
* Sales Trend Analysis Report
